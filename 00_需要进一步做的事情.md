## 📋 **自适应推荐引擎的核心内容**

基于代码分析，自适应推荐引擎包含以下主要组成部分：

### **🧠 1. 核心推荐算法**
- **VKS测试权重系统**: 基于用户的VKS(Vocabulary Knowledge Scale)测试结果，为不同学习模块分配权重
- **学习模式分析**: 分析用户的学习效率、准确率和偏好模块
- **智能推荐决策**: 按优先级检查紧急复习 → 常规复习 → 新内容学习

### **⏰ 2. 间隔重复算法 (SRS)**
- **改进的SM-2算法**: 基于SuperMemo-2算法的改进版本
- **动态间隔调整**: 根据用户表现动态调整复习间隔
- **质量因子计算**: 综合考虑正确性、响应时间、信心度、犹豫次数

### **📊 3. 用户行为分析**
- **学习模式识别**: 
  - 效率等级: `high` (>1.2), `medium` (0.8-1.2), `low` (<0.8)
  - 准确率等级: `high` (>85%), `medium` (70-85%), `low` (<70%)
- **学习偏好分析**: 基于各模块学习时间分布确定偏好
- **掌握程度追踪**: 实时更新词汇掌握水平

### **🎯 4. 推荐类型**
```python
推荐类型包括:
- urgent_review: 紧急复习 (掌握程度下降的词汇)
- scheduled_review: 计划复习 (到期复习的词汇) 
- new_learning: 新内容学习 (推荐新词汇)
- explore_learning: 探索学习 (默认推荐)
```

### **🏗️ 5. 数据库模型**
- **UserProfile**: 用户基础信息
- **LearningSession**: 学习会话记录
- **ExerciseRecord**: 练习记录详情
- **UserProgress**: 用户学习进度
- **AdaptiveRecommendation**: 推荐记录
- **ReviewSchedule**: 复习计划

### **📈 6. API接口**
```python
核心API端点:
- GET  /api/adaptive/recommendation/<user_id> - 获取个性化推荐
- POST /api/adaptive/feedback - 记录推荐反馈
- GET  /api/review/user/<user_id>/due - 获取到期复习内容
- GET  /api/analytics/user/<user_id>/dashboard - 获取学习分析
```

### **⚙️ 7. 算法参数**
```python
VKS权重配置:
- A级: 字符(1.0) > 词汇(0.8) > 搭配(0.6) > 句子(0.4)
- B级: 词汇(1.0) > 搭配(0.8) > 句子(0.6) > 字符(0.2)
- C级: 搭配(1.0) > 句子(0.8) > 词汇(0.3) > 字符(0.1)
- D级: 句子(1.0) > 搭配(0.4) > 词汇(0.2) > 字符(0.0)

间隔重复参数:
- 基础间隔: [1, 3, 7, 14, 30, 90, 180, 365] 天
- 默认ease factor: 2.5 (范围: 1.3-5.0)
- 掌握阈值: 0.8
```

### **🔄 8. 自适应机制**
- **实时调整**: 基于用户表现实时调整推荐策略
- **反馈循环**: 收集用户反馈优化推荐效果
- **个性化路径**: 为每个用户生成独特的学习路径
- **智能复习**: 根据遗忘曲线安排最优复习时间
