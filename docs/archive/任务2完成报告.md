### 3. 后端API端点 (`app_phase2.py`)

新增API端点:

#### 3.1 注册API
```
POST /api/auth/register
{
  "username": "test_user",
  "password": "test123456",
  "email": "test@example.com",
  "native_language": "English"
}
```

#### 3.2 登录API
```
POST /api/auth/login
{
  "username": "test_user",
  "password": "test123456",
  "remember_me": false
}
```
- 返回session_token
- 设置HttpOnly cookie

#### 3.3 登出API
```
POST /api/auth/logout
Headers: X-Session-Token: <token>
```

#### 3.4 获取当前用户
```
GET /api/auth/me
Headers: X-Session-Token: <token>
```

#### 3.5 验证Session
```
GET /api/auth/validate
Headers: X-Session-Token: <token>
```

#### 3.6 修改密码
```
POST /api/auth/change-password
Headers: X-Session-Token: <token>
{
  "old_password": "oldpass",
  "new_password": "newpass"
}
```

**认证方式**:
- ✅ Header: `X-Session-Token`
- ✅ Cookie: `session_token`
- ✅ 支持两种方式并存

## 🔧 技术实现

### 密码加密
```python
from werkzeug.security import generate_password_hash, check_password_hash

# 加密
password_hash = generate_password_hash(password, method='pbkdf2:sha256')

# 验证
check_password_hash(password_hash, password)
```

### Session Token生成
```python
import secrets

session_token = secrets.token_urlsafe(32)  # 安全随机字符串
```

### 数据库Schema
```sql
-- user_profile 表新增字段
ALTER TABLE user_profile ADD COLUMN password_hash TEXT;
ALTER TABLE user_profile ADD COLUMN email TEXT;
ALTER TABLE user_profile ADD COLUMN last_login DATETIME;

-- user_session 表
CREATE TABLE user_session (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id TEXT NOT NULL,
    session_token TEXT UNIQUE NOT NULL,
    created_at DATETIME NOT NULL,
    expires_at DATETIME NOT NULL,
    is_active BOOLEAN DEFAULT 1,
    updated_at DATETIME,
    device_info TEXT,
    ip_address TEXT
);
```

## 🎯 使用流程

### 新用户注册流程
```
1. 访问 /register
2. 填写注册表单
3. 提交 → POST /api/auth/register
4. 注册成功 → 跳转到登录页
5. 输入用户名密码
6. 提交 → POST /api/auth/login
7. 获取session_token
8. 保存到localStorage和Cookie
9. 跳转到首页
```

### 老用户登录流程
```
1. 访问 /login
2. 输入用户名/邮箱和密码
3. (可选) 勾选"记住我"
4. 提交 → POST /api/auth/login
5. 获取session_token
6. 保存到localStorage和Cookie
7. 跳转到首页
```

### 前端认证检查
```typescript
// 检查是否登录
const sessionToken = localStorage.getItem('session_token');

// 验证token
const response = await fetch('/api/auth/validate', {
  headers: { 'X-Session-Token': sessionToken }
});

const result = await response.json();
if (!result.valid) {
  // 跳转到登录页
  router.push('/login');
}
```
