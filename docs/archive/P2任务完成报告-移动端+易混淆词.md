## ✅ 任务2: 易混淆词辨析

### 交付物清单

#### 1. 数据库设计
**新增表**:
```sql
-- 易混淆词对表
CREATE TABLE confusable_pairs (
    id INTEGER PRIMARY KEY,
    word1_id INTEGER,
    word2_id INTEGER,
    reason TEXT,           -- 混淆原因
    difference TEXT,       -- 详细区别
    examples TEXT,         -- 例句对比
    tips TEXT,            -- 记忆技巧
    difficulty_level INTEGER
);

-- 练习记录表
CREATE TABLE confusable_exercise_records (
    id INTEGER PRIMARY KEY,
    user_id VARCHAR(100),
    pair_id INTEGER,
    is_correct BOOLEAN,
    response_time INTEGER
);
```

#### 2. 后端API (`confusable_api.py`)
**API端点**:
- `GET /api/confusable/pairs` - 获取词对列表
- `GET /api/confusable/pair/<id>` - 获取单个词对详情
- `POST /api/confusable/exercise/record` - 记录练习
- `GET /api/confusable/user/<id>/statistics` - 用户统计

**代码统计**: 约200行Python
