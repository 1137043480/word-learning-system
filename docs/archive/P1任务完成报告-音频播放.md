  - 播放/暂停

  - 播放结束回调

- 🎨 **UI状态**
  - 加载状态显示
  - 播放状态指示
  - 错误状态提示

- ⚙️ **可配置选项**
  - 语言设置（默认zh-CN）
  - 按钮大小（sm/md/lg）
  - 语速调整（0.9倍速，更清晰）
  - 音量和音调控制

**技术实现**:
```typescript
// 使用Web Speech API
const utterance = new SpeechSynthesisUtterance(text);
utterance.lang = 'zh-CN';
utterance.rate = 0.9;
window.speechSynthesis.speak(utterance);

// 支持音频文件
const audio = new Audio(audioUrl);
audio.play();
```

## 🔧 技术实现

### Web Speech API

**优势**:
- ✅ 浏览器原生支持
- ✅ 免费，无需API密钥
- ✅ 无需音频文件
- ✅ 支持多种语言
- ✅ 实时生成

**局限性**:
- ⚠️ 音质依赖浏览器
- ⚠️ 不同浏览器效果不同
- ⚠️ 需要网络（部分浏览器）

**浏览器支持**:
- ✅ Chrome/Edge (最佳)
- ✅ Safari (良好)
- ✅ Firefox (可用)
- ❌ IE (不支持)

### 扩展能力

支持三种音频源：
1. **Web Speech API** (当前实现)
2. **预录音频文件** (已支持接口)
3. **外部TTS服务** (可扩展)

```typescript
<AudioPlayer 
  text="发生"           // TTS文本
  audioUrl="/audio/1.mp3"  // 可选：音频文件
  language="zh-CN"     // 语言
  buttonSize="md"      // 按钮大小
/>
```

## 💡 使用示例

### 基础用法
```typescript
import AudioPlayer from '@/components/AudioPlayer';

<AudioPlayer 
  text="发生" 
  language="zh-CN"
/>
```

### 高级用法
```typescript
<AudioPlayer 
  text="不愿意发生的事情终于出现了。"
  language="zh-CN"
  audioUrl="/audio/example1.mp3"  // 可选
  buttonSize="lg"
  autoPlay={false}
  onPlay={() => console.log('播放开始')}
  onEnd={() => console.log('播放结束')}
  onError={(error) => console.error('播放错误', error)}
/>
```

### 快捷组件
```typescript
// 小按钮
<QuickAudioButton text="发生" />

// 自动播放（无按钮）
<AutoPlayAudio text="发生" />
```
