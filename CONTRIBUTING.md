# Contributing to Adaptive Chinese Vocabulary Learning System

Thank you for your interest in contributing! This project welcomes contributions from the community.

## How to Contribute

### Reporting Bugs
- Open an issue with the `[Bug]` prefix
- Include steps to reproduce, expected vs actual behavior

### Suggesting Features
- Open an issue with the `[Feature]` prefix
- Describe the use case and proposed solution

### Submitting Code
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Commit with [Conventional Commits](https://www.conventionalcommits.org/) format
4. Push and open a Pull Request

## Development Setup

```bash
# Backend
pip install flask flask-cors flask-sqlalchemy
python app_phase2.py

# Frontend
npm install
npm run dev
```

## Code Style
- **TypeScript/React**: Functional components, 2-space indent, PascalCase files
- **Python**: PEP 8 compliant
